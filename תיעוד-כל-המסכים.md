### 2. SignIn.jsx — התחברות

**📍 מסלול:** `/login`
**🎯 מטרה:** כניסה למערכת עם אימייל + סיסמה + בחירת תפקיד.

**שדות טופס:**
- אימייל
- סיסמה (עם אייקון עין)
- בחירת תפקיד (לקוח / מקצוען / אדמין)
- "זכור אותי"
- "שכחת סיסמה"

**🔌 API:** `POST http://localhost:8080/api/auth/login`

**ניווט לפי תפקיד:**
- `CLIENT` → `/client/dashboard`
- `ADMIN` → `/admin`
- `PROFESSIONAL` → `/pro/dashboard`

**שמירת פרטים:**
```jsx
localStorage.setItem('token', data.token);
localStorage.setItem('role', data.role);
localStorage.setItem('fullName', data.fullName);
```

### 3. SignUp.jsx — הרשמה

**📍 מסלול:** `/register`
**🎯 מטרה:** טופס הרשמה בן 4 שלבים.

**4 שלבי ההרשמה:**

| שלב | תוכן | למי |
|------|------|-----|
| **1** | בחירת תפקיד (לקוח/מקצוען) | לכולם |
| **2** | פרטים אישיים (שם, אימייל, סיסמה, טלפון, כתובת) | לכולם |
| **3** | פרטי מקצוע (קטגוריות, רדיוס, מחיר, ביו) | **רק מקצוענים** |
| **4** | מסך הצלחה ✅ | לכולם |

**🔌 API:** `POST http://localhost:8080/api/auth/register`

**תכונות מיוחדות:**
- סרגל התקדמות דינמי
- אנימציית רעידה בשגיאה
- העלאת תמונת פרופיל (עד 5MB)

## 👤 מסכי לקוח

### 4. ClientDashboard.jsx — דשבורד לקוח

**📍 מסלול:** `/client/dashboard`
**🎯 מטרה:** הדף הראשי של הלקוח — הזמנות פעילות, פעולות מהירות, התראות.

**חלקי המסך:**
- **ניווט עליון** — לוגו, פעמון התראות, פרופיל, יציאה
- **ברכה אישית** — "שלום, [שם] 👋"
- **3 כרטיסי פעולה:**
  - 📷 Snap (צילום תקלה)
  - 🔍 Search (חיפוש מקצוען)
  - 🗺️ Idea Map (מפת רעיונות)
- **רשימת הזמנות פעילות** עם כפתורי פעולה לפי סטטוס
- **3 מודלים:** עריכה, מעקב, ביטול

**🔌 API:** `GET http://localhost:8080/api/client/bookings`

**סטטוסים:**
| סטטוס | צבע | כפתורים |
|--------|------|---------|
| `pending` | 🟠 כתום | בטל / ערוך |
| `confirmed` | 🔵 כחול | צור קשר / בטל |
| `in_progress` | 🟢 ירוק | עקוב |
| `completed` | ⚪ אפור | ⭐ דרג |
| `cancelled` | 🔴 אדום | - |

**ביטול הזמנה:**
- פחות מ־48 שעות → **דמי ביטול 50₪**
- מעל 48 שעות → **חינם**

### 5. SnapAnIssue.jsx — צילום תקלה עם AI

**📍 מסלול:** `/client/snap`
**🎯 מטרה:** צ'אט עם AI שמאבחן תקלות מתמונה או מתיאור.

**תהליך:**
1. המשתמש מעלה תמונה או מקליד תיאור
2. ה־AI מנתח ומחזיר:
   - **DIY Card** — אם אפשר לתקן בעצמך (צעדים, כלים, זמן, עלות)
   - **Pro Required** — אם צריך מקצוען (אזהרה + כפתור חיפוש)

**🔌 API:** `analyzeIssue({ text, imageBase64 })` — שירות AI

**States:**
- `messages` — היסטוריית צ'אט
- `input` — טקסט הקלדה
- `imagePreview`, `imageBase64` — תמונה
- `loading` — בזמן ניתוח
- `lastResult` — התשובה האחרונה

**פיצ'רים:**
- כפתורי בחירה מהירה (קטגוריות נפוצות)
- העלאת תמונה מהמצלמה או מהקבצים
- תצוגת צ'אט עם אווטאר

### 6. BookaPro.jsx — הזמנת מקצוען

**📍 מסלול:** `/client/book-a-pro`
**🎯 מטרה:** הזמנת מקצוען ב־3 שלבים.

**3 שלבי ההזמנה:**

| שלב | תוכן |
|------|------|
| **1** | צילום/תיאור בעיה → AI מזהה קטגוריה |
| **2** | בחירת עיר + תאריך + שעה |
| **3** | בחירת מקצוען מרשימה מאומתת |

**🔌 API:** `GET /api/pros/search?specialty={cat}`

**States עיקריים:**
- `step` — שלב 1-3
- `cat`, `issue` — קטגוריה וסוג בעיה
- `messages`, `chatInput` — צ'אט AI
- `city`, `date`, `time` — פרטי הזמנה
- `results`, `modal` — רשימת מקצוענים

**פיצ'רים:**
- אבחון AI אוטומטי של הבעיה
- דירוגים + מחירים של כל מקצוען
- תנאי ביטול מוצגים בבירור
